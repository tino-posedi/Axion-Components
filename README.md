# Axion Components

Axion Components is a collection of modern, flat design UI components developed in Python. It is primarily intended for the development of industrial software applications.

All components follow a consistent flat design language, giving them a clean and contemporary appearance. In addition to aesthetics, strong emphasis has been placed on functionality, usability, and reliability to meet the practical demands of professional software environments.

## Component Overview  

| Component        | Description |
|------------------|-------------|
| **ChannelPlot**      | Customizable multichannel realtime plotting widget, featuring buffered time window visualization, peak/average overlays, and light/dark theme support. |
| **Expander**      | Lightweight collapsible expander panel with a gradient header, hover effects, and one click toggle to show/hide embedded content. |
| **FlatButton**      | Modern flat button with optional gradients, soft shadow, and smooth hover/press animations via color/offset blending. |
| **ProgressBar**      | Flexible progress bar supporting segmented or smooth fill modes, horizontal/vertical orientations, and optional gradient styling. |
| **RibbonToolbar**      | Microsoft Office style ribbon with themed tabs, grouped controls, smooth expand/collapse animation, and optional auto hide behavior. |
| **SimpleToolbar**      | Delphi style toolbar with icon buttons, optional gradient background, hover/pressed hot-tracking, and lightweight tooltips. |
| **StateButton**      | Flat toggle button that maintains an on/off state, supports hover/press styling, and allows dynamic border/text updates via callbacks. |
| **StaticPlot**      | Static trend chart widget that renders a sliding window line/band plot with grid/axes, optional 3D style depth, and color coded segment direction. |
| **StatusPanel**      | Compact status panel that displays a colored message badge with optional flashing (smooth or stepped) and automatic named color to hex support. |
| **TabbedNotebook**      | Clean, flat styled notebook that adds themed tabs and easy fixed size page creation. |
